API Bosques

La API (acrónimo de Application Programming Interface), es un instrumento tecnológico que permite acceder a los recursos de información que genera y distribuye el Programa Nacional de Conservación de Bosques para la Mitigación del Cambio Climático (PNCBMCC) a través de la plataforma GEOBOSQUES.

Rest: stockBosquePerdidaNacional
Recurso
http://geobosques.minam.gob.pe/geobosque/ws/rest/BOSQUEPERDIDA/stockBosquePerdidaNacional
Descripción

Devuelve datos de stock de bosque, perdida de bosque, perdida por rangos a nivel nacional.

Credencial

No Aplica.

Parámetros Request

No Aplica.

Response
Nombre Tipo Dato Descripción
result Número Resultado de la operacion; devuelve 1 si es exito y -1 si se presentan errores
stock_bosque_ultimo Número Total de bosque, en hectareas, al ultimo año del monitoreo.
